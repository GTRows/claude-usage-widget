# Claude Usage Widget 1.13.0-gtrows.1

GTRows fork release that takes the `claude-usage` CLI cross-platform
via npm and lands a full set of keyboard shortcuts for the widget.

## Highlights

- **CLI on npm — one install across Windows, macOS, Linux.** The
  `claude-usage` CLI now publishes to npm on every tag. Prereleases
  ship under dist-tag `next`, stable under `latest`. The tarball is
  intentionally small (~25 kB) and carries only CLI sources plus
  shared helpers.
  ```bash
  npm install -g claude-usage-widget        # latest stable
  npm install -g claude-usage-widget@next   # latest prerelease
  claude-usage login --key <k> --org <o>
  claude-usage status
  ```
- **Keyboard shortcuts.** Window-focused, no global accelerators:
  - `Ctrl/Cmd + R` — Refresh usage
  - `Ctrl/Cmd + G` — Toggle usage graph
  - `Ctrl/Cmd + H` — Toggle history table
  - `Ctrl/Cmd + M` — Toggle compact mode
  - `Ctrl/Cmd + P` — Pin on top
  - `Ctrl/Cmd + ,` — Open / close settings
  - `Ctrl/Cmd + Q` — Quit the widget entirely
  Shortcuts skip editable targets, so typing in an input still works.
  The default Electron menu is disabled, so `Ctrl+R` no longer reloads
  the renderer mid-session.
- **Settings panel: `06 Shortcuts` section.** Read-only list of every
  shortcut rendered with mono keycaps (shows `⌘` on macOS, `Ctrl`
  everywhere else). Title-bar button tooltips now include the hint.

## Also in this release

- Roadmap now tracks two new items: desktop notifications for peak
  throttle window start / end, and keeping the widget window
  draggable while the settings panel is open.

## Install

Desktop app artifacts for your platform are below — Setup installer
for Windows, DMG for macOS (Intel / Apple Silicon), AppImage / RPM /
DEB / Pacman / tar.gz for Linux.

CLI only, from any OS with Node.js 18+:

```bash
npm install -g claude-usage-widget
```
