# Claude Usage Widget 1.12.0-gtrows.1

GTRows fork release focused on surfacing Anthropic's 2026-03-26 peak-hour
session throttle inside the widget, plus fixes to update-check and a
fuller Linux packaging matrix.

## Highlights

- **New: Peak-hour throttle indicator.** Anthropic's 2026-03-26 change
  drains 5-hour session limits faster during weekday 5-11 AM PT /
  8 AM-2 PM ET / 1-7 PM GMT (12:00-18:00 UTC, Mon-Fri) for Free, Pro,
  and Max plans. The widget now surfaces this in two places:
  - **Tray icon:** edge-to-edge red underline beneath the number
    whenever you are currently inside the peak window.
  - **Dashboard:** thin card under the Session / Weekly cards with a
    `PEAK` / `OFF-PEAK` chip, `Throttled - session drains faster` vs
    `Normal rate`, countdown to the next boundary, and the peak
    window translated to your local time zone.
- **Fix: Update check now sees fork prereleases.** The previous
  release relied on `/releases/latest`, which hides prereleases and
  missed v1.11.0-gtrows.1. The check now scans `/releases?per_page=20`
  and compares versions via the shared semver comparator, so fork
  tags surface reliably.
- **Linux packaging expanded.** In addition to AppImage, the release
  now ships RPM, DEB, Pacman, and tar.gz targets for both x64 and
  arm64.
- **CI hardening.** Tests no longer try to restore an npm cache that
  doesn't exist (package-lock.json is gitignored). macOS build is
  now manual-only until notarization secrets are configured.

## Install

Pick the artifact for your platform below. Windows users typically
want the Setup installer. On Linux: `.rpm` for Fedora/RHEL, `.deb`
for Ubuntu/Debian, `.pacman` for Arch, AppImage or `.tar.gz` for
anything else.
