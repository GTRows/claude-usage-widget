# Claude Usage Widget 1.13.0-gtrows.1

GTRows fork release that lands a full set of keyboard shortcuts for
the widget.

## Highlights

- **Keyboard shortcuts.** Window-focused, no global accelerators:
  - `Ctrl/Cmd + R` — Refresh usage
  - `Ctrl/Cmd + G` — Toggle usage graph
  - `Ctrl/Cmd + H` — Toggle history table
  - `Ctrl/Cmd + M` — Toggle compact mode
  - `Ctrl/Cmd + P` — Pin on top
  - `Ctrl/Cmd + ,` — Open / close settings
  - `Ctrl/Cmd + Q` — Quit the widget entirely
  Shortcuts skip editable targets, so typing in an input still works.
  The default Electron menu is disabled, so `Ctrl+R` no longer reloads
  the renderer mid-session.
- **Settings panel: `06 Shortcuts` section.** Read-only list of every
  shortcut rendered with mono keycaps (shows `⌘` on macOS, `Ctrl`
  everywhere else). Title-bar button tooltips now include the hint.

## Also in this release

- Roadmap now tracks two new items: desktop notifications for peak
  throttle window start / end, and keeping the widget window
  draggable while the settings panel is open.
